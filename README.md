# Death Note Chess

**Death Note Chess** — это не просто шахматная доска, а прототип продуктовой веб-платформы для шахмат в эстетике notebook noir / Death Note. Идея проекта: превратить обычную партию в атмосферную дуэль, где игрок не только ходит фигурами, но и прокачивает рейтинг, участвует в турнирах, получает разбор от AI Coach и открывает premium-контент.

Проект сделан как MVP сервиса, который можно развивать в полноценную шахматную платформу с монетизацией, соревнованиями и социальным слоем.

## Для кого это

Платформа рассчитана на:

- игроков, которым скучны обычные шахматные сайты;
- новичков, которым нужен понятный AI Coach после партии;
- друзей, которые хотят быстро сыграть по коду комнаты;
- организаторов мини-турниров на 32 участника;
- пользователей, которым важны рейтинг, прогресс, темы, кастомные фигуры и соревновательный режим.

## Почему это ценно

Большинство учебных шахматных проектов ограничиваются доской 8x8. Здесь сделан продуктовый слой: retention, premium-планы, сезонные турниры, зрительский режим, заявки участников, профили игроков, shareable champion card и пост-турнирный отчет.

Death Note тематика делает проект запоминающимся, а шахматная логика, AI Coach и турнирная система дают реальную пользу пользователю.

## Что реализовано

- Полноценные шахматные правила через `chess.js`: шах, мат, пат, рокировка, взятие на проходе, превращение пешки и ничьи.
- Игра против локального ИИ на negamax-алгоритме.
- Игра вдвоем на одном экране.
- Онлайн-комнаты по коду через `localStorage` как демо WebSocket-ready механики.
- AI Coach после партии: оценка точности, поиск критического хода и подсказка лучшей альтернативы.
- Кастомные noir-фигуры в стиле Death Note.
- Сохранение прогресса, рейтинга, темы, плана и статистики в `localStorage`.
- Мультиязычный интерфейс: русский, английский, казахский и китайский.
- Адаптивный дизайн для desktop и mobile.

## Kira Cup: турнирный режим

Отдельный фокус проекта — соревновательный режим **Kira Cup**:

- сетка турнира на 32 игрока;
- ручной ввод имен участников;
- выбор первого соперника;
- подключение к турниру по коду;
- Spectator Mode: зритель может открыть турнир только для просмотра;
- Tournament Lobby: участники могут подать заявку по коду;
- организатор принимает или отклоняет заявки;
- Match Result Submission: счет матча вводится как `1-0`, `0-1` или `1/2`;
- Player Profiles: город, рейтинг, победы, форма и любимый дебют;
- Post-Tournament Report: MVP, главный upset, лучший матч и изменение рейтинга;
- Shareable Result Card: красивая карточка чемпиона `Kira Cup Champion`.

## Бизнес-логика

В проект встроен прототип монетизации:

- Free / Pro / Elite планы;
- locked-состояния для premium-фич;
- Upgrade to Pro CTA;
- premium темы доски;
- Elite Skin Vault;
- Pro-доступ к турнирам, AI Coach reports и online-функциям;
- позиционирование кастомных фигур и финальных эффектов как платного контента.

Это показывает, что проект задуман не как “курсовая работа”, а как основа сервиса с потенциальной выручкой и возвращаемостью пользователей.

## Локальный запуск

```bash
npm install
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:5173/
```

## Production build

```bash
npm run build
```

Папка для деплоя:

```text
dist
```

## Как задеплоить

### Vercel

1. Загрузите проект в GitHub.
2. Откройте [vercel.com](https://vercel.com).
3. Создайте новый проект из GitHub-репозитория.
4. Укажите настройки:
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. Нажмите Deploy.

### Netlify

1. Загрузите проект в GitHub.
2. Откройте [netlify.com](https://netlify.com).
3. Создайте новый site from Git.
4. Укажите настройки:
   - Build Command: `npm run build`
   - Publish Directory: `dist`
5. Нажмите Deploy.

## Коротко

**Death Note Chess** — это шахматная веб-платформа с атмосферой, AI Coach, турнирами, premium-контентом и продуктовой логикой. Проект демонстрирует не только умение реализовать шахматные правила, но и способность думать как создатель сервиса: про пользователя, удержание, монетизацию и масштабирование.
